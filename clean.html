WeCleanCity - Clean India Maker

Project Description

WeCleanCity is a web-based application aimed at promoting cleanliness and civic responsibility in urban and rural areas of India. The platform allows citizens to actively participate in maintaining a clean environment by reporting areas that need cleaning, tracking cleanup drives, and raising awareness about sanitation and hygiene. The project aligns with the vision of “Clean India” by empowering users to contribute directly to their communities.

Technology Overview

Frontend: HTML, CSS, JavaScript

Backend: Python Flask

Database: MySQL

Key Features

  • User registration and login for secure access.
  • Reporting dirty or unhygienic areas with details and optional images.
  • Viewing and tracking the status of reported complaints.
  • Admin panel to monitor reports, assign cleanup tasks, and manage users.
  • Interactive dashboard to visualize the progress of cleanliness drives.
  • Notifications and reminders for upcoming cleanup events.

Impact

By integrating technology with social responsibility, WeCleanCity empowers citizens to actively participate in creating cleaner and healthier surroundings. It fosters community engagement, improves public hygiene, and contributes to the larger goal of a Clean India Mission.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python Flask
  • Database: MySQL
  • Other Tools: Bootstrap, AJAX, Flask-Login
